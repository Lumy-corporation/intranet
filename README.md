🛠️ Spécifications Fonctionnelles : Lumy Intranet

1. Système d'Authentification (SSO Google)
Accès Restreint : La connexion doit se faire exclusivement via le bouton "Se connecter avec Google".
Domaine Autorisé : Seuls les comptes possédant une adresse @lumycorp.com sont autorisés à accéder à la plateforme.

2. Gestion des Rôles et Accès
État "Sans Rôle" : Si un utilisateur se connecte mais n'a pas encore de rôle attribué, un message d'alerte s'affiche en plein écran sur sa page d'accueil.
Procédure d'Alerte : Un bouton doit permettre à l'utilisateur de signaler l'absence de rôle.
Automatisation des Mails : Ce bouton déclenche un mail automatique de no-reply@lumycorp.com vers support@lumycorp.com, précisant l'identité de l'utilisateur (ex: theo.rabillard@lumycorp.com) et le problème rencontré.

3. Contrôle du Site Web Public (Back-Office)
Gestion des Offres d'Emploi : La section "Offres de job" du site web public doit être pilotable directement depuis l'intranet (Ajout/Modification/Suppression).
Migration Centralisée : L'objectif final est de migrer progressivement tous les systèmes de Lumy Corp dans cet outil unique.

4. Base de Données des Membres (CRM Interne)
Registre des Adhérents : Tous les adhérents doivent être enregistrés et listés dans l'intranet.
Gestion des Profils : Chaque fiche membre contiendra ses informations personnelles et ses rôles spécifiques au sein de l'organisation.
